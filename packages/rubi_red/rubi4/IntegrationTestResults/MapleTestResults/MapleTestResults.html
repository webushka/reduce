Maple 13 Indefinite Integration Test Suite Results


The Maple 13 test suite results are available as text files. Each entry in these files is an integration problem for which Maple failed to get the optimal antiderivative. Each entry consists of the following items:
    1. a phrase indicating the type of deficiency;
    2. the integrand;
    3. the optimal antiderivative as contained in the test suite;
    4. the antiderivative as generated by Maple; and
    5. the leafcount size of the optimal vs. generated antiderivatives.

Rubi 3 Indefinite Integration Test Suite Results


The Rubi 3 test suite results are available as Portable Document Format (PDF) files and as Mathematica notebook files. Each entry in these files is an integration problem for which Rubi failed to get the optimal antiderivative. Each entry consists of the following items:
    1. a phrase indicating the type of deficiency;
    2. a list of the integrand, integration variable, number of rule applications desired, and number required;
    3. the optimal antiderivative as contained in the test suite; and
    4. the antiderivative as generated by Rubi.
